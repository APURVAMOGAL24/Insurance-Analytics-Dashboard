# 🛡️ InsuraSight — Insurance Claims & Policy Analytics Dashboard

## 📌 Overview
InsuraSight is a Power BI dashboard designed to analyze insurance policies, claims, and customer feedback.  
The dashboard provides insights into premium trends, claim distribution, policy status, and customer sentiment to support data-driven decision-making.

---

## 🎯 Objectives
- Analyze total premium, coverage, and claim amounts  
- Track claim status (Pending, Settled, Rejected)  
- Monitor active vs inactive policies  
- Understand customer feedback and sentiment  
- Identify trends across policy types and age groups  

---

## 📊 Key Insights
- Total Premium Amount: **5.98M**
- Total Coverage Amount: **600.55M**
- Total Claim Amount: **16.91M**
- Majority of policies are **Active (~75%)**
- Travel insurance contributes highest premium share  
- Claims are mostly **Rejected or Settled**  
- Adult customers contribute highest claim amount  

---

## 📈 Dashboard Features
- KPI Cards for quick business overview  
- Policy Type-wise premium analysis  
- Claim status distribution  
- Active vs Inactive policy breakdown  
- Age group-based claim analysis  
- Customer feedback sentiment (Word Cloud + Table)  

---

## 🛠️ Tools & Technologies
- Power BI  
- Microsoft Excel (Data Source)  
- Data Cleaning & Transformation (Power Query)  

---

## 📂 Dataset
- Source: Microsoft Excel  
- Contains:
  - Policy details  
  - Customer demographics  
  - Claim information  
  - Feedback & sentiment data  

---

## 🚀 How to Use

1. Download the `.pbix` file from this repository
2. Open it using Microsoft Power BI Desktop
3. If prompted, update the data source path
4. Refresh the data to view the latest insights.
